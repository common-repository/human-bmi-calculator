= Features =

  * Easy installation
  * Very Easy to use. Just copy the shortcode and paste it into the desired section.


== Installation ==
1. Upload `human-bmi-calculator` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
